# Tech Stack
- **Language:** SWIFT 5
- **IDE:** Xcode 14
- **iOS:** 16
- **Archetecture:** MVC
- **Frameworks:**  UIKit, Foundation
- **Layout:** Interface Builder
- **GCD:** Yes 
- **Features:** 
  - Saving data to Plist
  - DataModel
  - TableView
  - TapGestureRecognizer for cells
  - Delegates
  - Segues
  - Unwind
  - And many more
